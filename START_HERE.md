## Alternative: Use HTTPS Instead of SSH

If SSH isn't working for you, use this command instead:

```bash
cd /Users/mjh5153/Downloads/demo && git remote add origin https://github.com/YOUR_USERNAME/demo.git && git branch -M main && git push -u origin main
```

You'll be prompted to enter a Personal Access Token (not your password).

## Quick Troubleshooting

**SSH not working?**
→ Use HTTPS command above, OR
→ See "SSH Connection Issues" in PUSH_COMMAND.md

**Can't remember your GitHub username?**
→ Check GitHub.com/settings/profile

**"Remote already exists" error?**
→ Run: `git remote remove origin` first
